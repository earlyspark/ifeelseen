# I Feel Seen

A card-drawing experience that meets you exactly where you are.

Pick from three hidden piles and receive a personal reflection grounded in wisdom — not affirmations, not advice. Just truth.

**ifeelseen.ai**

## Tech Stack

- Next.js
- Tailwind CSS
- Framer Motion
- OpenAI API

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
